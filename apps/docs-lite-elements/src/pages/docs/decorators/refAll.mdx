# @ref
---

This decorator, akin to the querySelectorAll, enables you to search for an element within the component tree.

What sets it apart is its ability to search using the value of the data-ref attribute, eliminating the need to define it in the selector.

It accepts a selector and a property, which is then assigned to the found elements and returned as an array. Internally, it leverages the ref.

<CodeBlock lang='html'>{`<div data-ref="my-div">One</div>
<div data-ref="my-div">Two</div>`}</CodeBlock>

<CodeBlock lang='typescript'>{`@refAll("my-div") text: HTMLDivElement[]; `}</CodeBlock>
