# @querySelector
---

This is a decorator that allows you to query for an element inside the component tree. 

It takes a selector and a property to assign the element to. Behind the scenes, it uses querySelector.

<CodeBlock lang='typescript'>{`@querySelector("[data-text]") text: HTMLElement; `}</CodeBlock>
