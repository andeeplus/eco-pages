# Installation
---

To get started with Eco Pages, you need to install the package in your project. You can do this by running the following command:

<CodeBlock lang='bash'>
{`bun install eco-pages`}
</CodeBlock>

## Usage

After installing the package, you can use the following commands to start developing your project:

<CodeBlock lang='json'>
{`{
  "scripts": {
	"dev": "eco-pages dev",
	"build": "eco-pages build",
	"start": "eco-pages start",
	"preview": "eco-pages preview"
}`}
</CodeBlock>
