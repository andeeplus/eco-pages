# Building a page

There are several you can use to build a page in Eco Pages, it mostly depends on the integrations you are using.

Currently the following integrations are supported:

- [`@eco-pages/kitajs`](/docs/integrations/kitajs)
- [`@eco-pages/lit`](/docs/integrations/lit)
- [`@eco-pages/mdx`](/docs/integrations/mdx)
- [`@eco-pages/react`](/docs/integrations/react)
