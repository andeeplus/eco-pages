# Building a page

There are several you can use to build a page in Eco Pages, it mostly depends on the integrations you are using.

Currently the following integrations are supported:

- [`@ecopages/kitajs`](/docs/integrations/kitajs)
- [`@ecopages/lit`](/docs/integrations/lit)
- [`@ecopages/mdx`](/docs/integrations/mdx)
- [`@ecopages/react`](/docs/integrations/react)
