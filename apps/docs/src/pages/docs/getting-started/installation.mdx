# Installation
---

To get started with Radiant, you need to install the package in your project. You can do this by running the following command:

<CodeBlock lang='bash'>
{`bun install @ecopages/radiant`}
</CodeBlock>

## Usage

After installing the package, you can use the following commands to start developing your project:

<CodeBlock lang='json'>
{`{
  "scripts": {
	"dev": "ecopages dev",
	"build": "ecopages build",
	"start": "ecopages start",
	"preview": "ecopages preview"
}`}
</CodeBlock>
